# 풀이
---
1부터 분해합을 계산하여 입력받은 결과와 같은 수를 출력하는 문제로 단순 for 문을 이용하여 풀어봤습니다.

```swift
let n = Int(readLine()!)!
var result = 0

for i in 1..<n {
  var sum = i
  var part = i
  
  // 각 자리수 더하기
  while part > 0 {
    sum += part % 10
    part /= 10
  }

  // 분해합이 입력받은 수와 같을 경우
  if n == sum {
    result = i
    break
  }
}

// 결과 출력
print(ans)
```

1. 자연수 n을 입력받습니다.
2. 1부터 n까지 분해합을 구합니다.
3. 분해합이 n과 같을 경우 for 문을 중단시킵니다.
